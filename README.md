# Procedurally Generated Game

A procedurally generated game built with C and Raylib.

## Setup

### macOS
```bash
brew install raylib
make run
```

### Linux
```bash
sudo apt-get install build-essential libraylib-dev
make run
```

## Build

```bash
make        # Build
make run    # Build and run
make debug  # Debug build
make clean  # Clean build artifacts
```
